常见问题解答

我可以试试变色龙吗？

该产品目前正在测试中，尚未公开发布。请联系我们以注册兴趣或请求演示 - 准备好后我们会与您联系。

它可以在没有互联网连接的情况下工作吗？

大多数功能都可以离线工作：编辑、会计、演示者和司机搬运工都在本地运行。 AI 功能需要云提供商（DeepSeek、OpenAI 等）的互联网连接，但可以使用本地 Ollama 或 LM Studio 实例离线工作。

支持哪些平台？

Linux 是主要的开发平台。 Windows 和 macOS 通过以下方式支持 platform_utils.py。 Android 配套应用程序通过 WebSocket 连接到桌面应用程序。

您可以添加功能或与我们的系统集成吗？

定制集成已在计划中。请联系我们告知您的需求 - 常见请求包括定制 AI 提供商、定制会计工作流程和 Tailscale ACL 自动化。条款有待最终确定。

API 密钥安全吗？

API 密钥存储在 api_keys.env 在您的本地计算机上，除了您选择的提供商之外，绝不会传输到任何地方。只读 DB API 服务器（端口 8769）仅限于您的 Tailscale 网络。
