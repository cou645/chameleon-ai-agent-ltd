🌍 域名配置

DNS 配置是大多数人感到困惑的部分——注册商登录、记录类型、传播等待。人工智能会尽可能直接处理它，并在其他地方提供准确的分步指导。

  • 自动或通过简短的调查问卷识别您的注册商
  • 在复制粘贴表中生成所需的确切 DNS 记录（A、AAAA、CNAME）
  • 提供特定于注册商的说明 — Squarespace、GoDaddy、Namecheap、Cloudflare、123-reg 等
  • 在您接触任何内容之前，对每种记录类型进行简单的英语解释
  • 通过 GitHub Pages API 设置自定义域 - 无需手动输入表单
  • 监控 DNS 传播并确认 HTTPS 证书颁发
  • 提供 API 的注册商完全自动化（Cloudflare、Namecheap）
⚠️
为什么这很重要： 在我们自己的测试中，域名配置引起了最大的混乱 - 域名悄悄地从 Google Domains 迁移到 Squarespace，没有明显的重定向，并且 DNS 记录类型（A、AAAA、CNAME）完全不熟悉。现在，人工智能会预先询问您的注册商并给出指示，而无需事先具备 DNS 知识。
