Urutan diagnosis terstruktur

Perangkat keras → status kernel → status driver → jaringan → penyimpanan. AI menjalankan setiap langkah, membaca hasil JSON, lalu memutuskan tindakan selanjutnya. Ia tidak pernah melewatkan langkah atau membuat asumsi.

🏥

Protokol perbaikan bedah

Non-destruktif terlebih dahulu (muat ulang modul, mulai ulang layanan). Bertahan sebelum mengambil risiko (konfigurasi snapshot). Verifikasi di setiap langkah. Setiap tindakan yang memerlukan reboot akan memicu Dead Man's Switch yang menyetel a @reboot melanjutkan skrip secara otomatis.

🌐

Perbaikan jarak jauh melalui Tailscale

Diagnosis dan perbaiki mesin di mana pun di jaring Tailscale Anda. AI mengeluarkan perintah pada mesin jarak jauh melalui jembatan terminal, membaca hasil dan melanjutkan — bahkan saat reboot menggunakan sistem pos pemeriksaan/resume.

Porter Pengandar USB

Profil perangkat USB apa pun → analisis protokolnya → buat driver Linux yang siap diinstal

Jalur A

Protokol standar

Tidak ada halaman vendor yang ditemukan. Menghasilkan aturan udev + konfigurasi modprobe. Tidak diperlukan kompilasi. Bekerja dengan HID, CDC, penyimpanan massal, dan kelas standar lainnya.

udevmodprobe
Jalur B

Protokol vendor parsial

Halaman vendor terdeteksi (≤2). Menghasilkan daemon hidraw/uhid userspace atau plugin ALSA. Berjalan di ruang pengguna — tidak diperlukan kompilasi kernel.

menyembunyikanplugin ALSA
Jalur C

Protokol berpemilik

Protokol yang sepenuhnya khusus. Menghasilkan modul kernel lengkap dengan kemasan DKMS dan dukungan kompilasi silang ARM untuk Raspberry Pi 3, 4 dan 5.

Modul kernelDKMSLENGAN

🖥️

Backend pengambilan lintas platform

ETW di Windows, IOKit di macOS, usbmon di Linux. Profil pada OS apa pun dan hasilkan driver Linux dari hasilnya profile.json.

🍓

Kompilasi silang Raspberry Pi

Target Makefile cross-aarch64 (Pi 4/5) dan cross-armv7 (Pi 3) disertakan secara otomatis di Jalur C. Bangun di x86 dan terapkan ke ARM.

📦

kemasan DKMS

Driver jalur C dikemas untuk DKMS sehingga dapat bertahan dalam pemutakhiran kernel tanpa kompilasi ulang manual. Ulasan AI analysis.json dan meminta rintisan TODO yang tersisa.
