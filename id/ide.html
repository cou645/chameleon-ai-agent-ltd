IDE yang didukung AI

Editor kode PyQt6 berfitur lengkap yang tidak hanya disarankan oleh AI — ia membaca, menulis, menjalankan, dan memverifikasi kode atas nama Anda melalui jembatan perintah terstruktur.

AI multi-penyedia 🎙️ Masukan suara & TTS 🔀 Tampilan berbeda ⚡ Respons streaming

Jembatan Komando AI

AI tidak hanya mengobrol — ia mengoperasikan editor melalui protokol JSON terstruktur

📖

Baca sekali, edit dengan bebas

AI membaca file satu kali dengan get_file_content, menyimpan konten dalam konteks, lalu mengeluarkan beberapa pengeditan berbasis jangkar tanpa membaca ulang di antara setiap perubahan. Tidak ada perjalanan pulang pergi yang sia-sia.

🎯

replace_text berbasis jangkar

Pengeditan merujuk pada teks yang tepat untuk ditemukan, bukan nomor baris. Nomor baris bergeser saat kode ditambahkan atau dihapus — jangkar tidak. Banyak replace_text panggilan dapat dikelompokkan dalam satu respons, semuanya diterapkan secara atom.

📝

write_file untuk penulisan ulang penuh

Untuk modul baru atau penulisan ulang lengkap, AI mengirimkan seluruh konten file dalam satu perintah. Sistem secara otomatis membuat a .bak cadangan sebelum menimpa dan membuka hasilnya di tab editor baru.

💻

Integrasi terminal

AI dapat menjalankan perintah shell dan menerima hasil JSON sebelum melanjutkan. Ia menunggu setiap hasil, menangani kode keluar, dan membedakan perintah diagnostik, konfigurasi, destruktif, dan mengakhiri sesi secara otomatis.

🛡️

Perlindungan injeksi

Semua konten yang dikembalikan dari file, terminal, dan halaman web dibungkus [UNTRUSTED_CONTENT] penanda. AI diinstruksikan untuk memperlakukan semua yang ada di dalamnya sebagai data — bukan instruksi — untuk mencegah serangan injeksi cepat dari konten file berbahaya.

Fitur Penyunting

🗂️

Editor kode multi-tab

Buka banyak file secara bersamaan. Setiap tab melacak riwayat pembatalannya sendiri, penyorotan sintaksis, dan konteks perintah AI.

🔀

Tampilan perbedaan bawaan

Satu klik untuk membandingkan status editor saat ini dengan versi terakhir yang disimpan. Perbedaan terpadu berkode warna dengan penyorotan tambah/hapus. Keluar dari mode diff untuk kembali ke pengeditan penuh.

💬

Streaming obrolan AI

Tanggapan mengalirkan token demi token ke dalam panel obrolan. Perintah IDE diekstraksi dan dieksekusi di thread latar belakang sehingga UI tetap responsif.

🗃️

Sistem prompt modular

Perintah sistem adalah file modul teks biasa ~/.chameleon/prompts/. Tumpuk kombinasi apa pun: Perbaikan Teknologi + Akademik + Obrolan. Jumlah token langsung dan perkiraan biaya per penyedia.

📜

DB riwayat perintah

Setiap perintah AI dicatat history.db dengan ID sesi, stempel waktu dan hasil. Jejak audit penuh untuk setiap perubahan file yang dibuat AI.

🌳

Peramban berkas

Panel pohon proyek untuk navigasi cepat. AI menggunakan list_project_files Dan list_open_files untuk menemukan jalur sebenarnya — ia tidak pernah menemukan lokasi file.

Suara & Ucapan

🎙️

Bisikan ucapan-ke-teks

Bisikan Lokal (whisper.cpp) mentranskripsikan suara Anda ke dalam masukan obrolan. Pengodean bebas genggam — mendiktekan pertanyaan, perintah, atau seluruh perintah tanpa menyentuh keyboard.

berbisik.cppInferensi lokalTidak ada awan STT
🔊

Piper teks-ke-ucapan

Respons AI dibacakan dengan lantang menggunakan Piper TTS — suara yang cepat, lokal, dan terdengar alami. Ideal untuk aksesibilitas atau bekerja dengan tangan sibuk. Jalur platform diselesaikan secara otomatis di Linux, Windows, dan macOS.

Stasiun Komando Perusahaan (CCS)

Jalankan Chameleon IDE sebagai hub kantor untuk armada Logistik 4×4 Anda — tidak diperlukan perangkat lunak pengiriman terpisah

🗺

Dorong rute & POI ke pengemudi

Kirim a .gpx file dari IDE langsung ke telepon pengemudi mana pun melalui saluran rekan. Rute dimuat secara instan di peta langsung pengemudi. Kumpulan POI mandiri — pemberhentian bahan bakar, bahaya, alamat pengiriman — dapat didorong secara independen dari rute mana pun dan muncul sebagai pin berkode warna.

📦

Pengiriman manifes & faktur

Buat manifes kargo atau siapkan faktur di IDE, lalu kirimkan secara peer-to-peer ke telepon pengemudi. Pengemudi melihat manifes yang dimuat sebelumnya — tanpa pemindaian, tanpa entri manual. Faktur dapat ditandatangani di layar dan dikembalikan melalui saluran yang sama.

🚨

Siaran ticker seluruh armada

Kirim peringatan prioritas — Darurat, Penting, Petunjuk, atau Info — dan peringatan tersebut akan muncul sebagai bilah ticker lebar penuh di setiap layar pengemudi secara bersamaan. Kantor tetap memegang kendali penuh atas perhatian armada tanpa perlu menelepon setiap kendaraan.

📡

Server sinyal internal — tanpa relai cloud

Server sinyal Bunglon (chameleon_signal.py) berjalan langsung di PC kantor bersama IDE, atau di dalam Termux di ponsel Android mana pun di armada. Ponsel armada terhubung melalui Tailscale atau hotspot lokal — tanpa relay pihak ketiga, tanpa tagihan server bulanan.
