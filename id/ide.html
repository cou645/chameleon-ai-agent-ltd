The AI Command Bridge

AI doesn't just chat — it operates the editor through a structured JSON protocol

📖

Baca sekali, edit dengan bebas

The AI reads a file once with get_file_content, menyimpan konten dalam konteks, lalu mengeluarkan beberapa pengeditan berbasis jangkar tanpa membaca ulang di antara setiap perubahan. Tidak ada perjalanan pulang pergi yang sia-sia.

🎯

replace_text berbasis jangkar

Pengeditan merujuk pada teks yang tepat untuk ditemukan, bukan nomor baris. Nomor baris bergeser saat kode ditambahkan atau dihapus — jangkar tidak. Banyak replace_text panggilan dapat dikelompokkan dalam satu respons, semuanya diterapkan secara atom.

📝

write_file untuk penulisan ulang penuh

For new modules or complete rewrites the AI sends the entire file content in one command. The system automatically creates a .bak cadangan sebelum menimpa dan membuka hasilnya di tab editor baru.

💻

Integrasi terminal

The AI can run shell commands and receive the JSON result before continuing. It waits for each result, handles exit codes, and distinguishes diagnostic, configuration, destructive and session-ending commands automatically.

🛡️

Perlindungan injeksi

Semua konten yang dikembalikan dari file, terminal, dan halaman web dibungkus [UNTRUSTED_CONTENT] markers. The AI is instructed to treat everything inside as data — never instructions — preventing prompt-injection attacks from malicious file content.

Fitur Penyunting

🗂️

Editor kode multi-tab

Open multiple files simultaneously. Each tab tracks its own undo history, syntax highlighting and AI command context.

🔀

Tampilan perbedaan bawaan

One click to compare the current editor state against the last saved version. Colour-coded unified diff with add/remove highlighting. Exit diff mode to return to full editing.

💬

Streaming AI chat

Tanggapan mengalirkan token demi token ke dalam panel obrolan. Perintah IDE diekstraksi dan dieksekusi di thread latar belakang sehingga UI tetap responsif.

🗃️

Sistem prompt modular

System prompts are plain-text module files in ~/.chameleon/prompts/. Stack any combination: Tech Repair + Academic + Chat. Live token count and cost estimate per provider.

📜

DB riwayat perintah

Every AI command is logged to history.db with session ID, timestamp and result. Full audit trail for every file change the AI made.

🌳

Peramban berkas

Project tree panel for fast navigation. The AI uses list_project_files Dan list_open_files untuk menemukan jalur sebenarnya — ia tidak pernah menemukan lokasi file.

Suara & Ucapan

🎙️

Whisper speech-to-text

Local Whisper (whisper.cpp) transcribes your voice into the chat input. Hands-free coding — dictate questions, commands or whole prompts without touching the keyboard.

Stasiun Komando Perusahaan (CCS)

Jalankan Chameleon IDE sebagai hub kantor untuk armada Logistik 4×4 Anda — tidak diperlukan perangkat lunak pengiriman terpisah

🗺

Dorong rute & POI ke pengemudi

Kirim a .gpx file dari IDE langsung ke telepon pengemudi mana pun melalui saluran rekan. Rute dimuat secara instan di peta langsung pengemudi. Kumpulan POI mandiri — pemberhentian bahan bakar, bahaya, alamat pengiriman — dapat didorong secara independen dari rute mana pun dan muncul sebagai pin berkode warna.

📦

Pengiriman manifes & faktur

Buat manifes kargo atau siapkan faktur di IDE, lalu kirimkan secara peer-to-peer ke telepon pengemudi. Pengemudi melihat manifes yang dimuat sebelumnya — tanpa pemindaian, tanpa entri manual. Faktur dapat ditandatangani di layar dan dikembalikan melalui saluran yang sama.

🚨

Siaran ticker seluruh armada

Kirim peringatan prioritas — Darurat, Penting, Petunjuk, atau Info — dan peringatan tersebut akan muncul sebagai bilah ticker lebar penuh di setiap layar pengemudi secara bersamaan. Kantor tetap memegang kendali penuh atas perhatian armada tanpa perlu menelepon setiap kendaraan.

📡

Server sinyal internal — tanpa relai cloud

Server sinyal Bunglon (chameleon_signal.py) runs directly on the office PC alongside the IDE, or inside Termux on any Android phone in the fleet. Fleet phones connect to it over Tailscale or the local hotspot — no third-party relay, no monthly server bill.
