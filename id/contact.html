Pertanyaan yang Sering Diajukan

Bolehkah saya mencoba Bunglon?

Produk ini sedang dalam pengujian dan belum tersedia untuk rilis publik. Hubungi kami untuk mendaftarkan minat atau meminta demo — kami akan menghubungi Anda jika sudah siap.

Apakah ini berfungsi tanpa koneksi internet?

Sebagian besar fitur bekerja secara offline: editor, akuntansi, presenter, dan porter driver semuanya dijalankan secara lokal. Fitur AI memerlukan koneksi internet untuk penyedia cloud (DeepSeek, OpenAI, dll.) tetapi bekerja secara offline dengan instance Ollama atau LM Studio lokal.

Platform mana yang didukung?

Linux adalah platform pengembangan utama. Windows dan macOS didukung melalui platform_utils.py. Aplikasi pendamping Android terhubung melalui WebSocket ke aplikasi desktop.

Bisakah Anda menambahkan fitur atau mengintegrasikannya dengan sistem kami?

Integrasi khusus direncanakan. Hubungi kami untuk menyampaikan kebutuhan Anda — permintaan umum mencakup penyedia AI khusus, alur kerja akuntansi yang dipesan lebih dahulu, dan otomatisasi ACL Tailscale. Persyaratan tunduk pada finalisasi.

Apakah kunci API aman?

Kunci API disimpan di api_keys.env di mesin lokal Anda dan tidak pernah dikirim ke mana pun selain penyedia yang Anda pilih. Server DB API baca-saja (port 8769) dibatasi untuk jaringan Tailscale Anda.
