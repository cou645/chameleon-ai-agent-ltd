Întrebări frecvente

Pot să încerc Chameleon?

Produsul este în prezent în testare și nu este încă disponibil pentru lansare publică. Contactați-ne pentru a vă înregistra interesul sau pentru a solicita o demonstrație — vă vom contacta când va fi gata.

Funcționează fără conexiune la internet?

Cele mai multe funcții funcționează offline: editorul, contabilitatea, prezentatorul și portarul șoferului rulează toate local. Funcțiile AI necesită o conexiune la internet pentru furnizorii de cloud (DeepSeek, OpenAI etc.), dar funcționează offline cu o instanță locală Ollama sau LM Studio.

Ce platforme sunt acceptate?

Linux este platforma principală de dezvoltare. Windows și macOS sunt acceptate prin platform_utils.py. Aplicația însoțitoare Android se conectează prin WebSocket la aplicația desktop.

Puteți adăuga o caracteristică sau puteți integra cu sistemul nostru?

Sunt planificate integrări personalizate. Contactați-ne cu cerințele dvs. — solicitările obișnuite includ furnizori personalizați de AI, fluxuri de lucru de contabilitate personalizate și automatizarea ACL Tailscale. Condiții supuse definitivării.

Sunt sigure cheile API?

Cheile API sunt stocate în api_keys.env pe mașina dvs. locală și nu a fost transmisă niciodată în altă parte decât furnizorul pe care îl alegeți. Serverul DB API numai pentru citire (portul 8769) este limitat la rețeaua dvs. Tailscale.

Trimite-ne un e-mail și te vom contacta în termen de o zi lucrătoare.
