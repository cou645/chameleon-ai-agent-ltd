Foire aux questions

Puis-je essayer Caméléon ?

Le produit est actuellement en test et n'est pas encore disponible pour une diffusion publique. Contactez-nous pour manifester votre intérêt ou demander une démo – nous vous contacterons lorsqu’elle sera prête.

Est-ce que ça marche sans connexion internet ?

La plupart des fonctionnalités fonctionnent hors ligne : le rédacteur, la comptabilité, le présentateur et le chauffeur-porteur fonctionnent tous localement. Les fonctionnalités d'IA nécessitent une connexion Internet pour les fournisseurs de cloud (DeepSeek, OpenAI, etc.) mais fonctionnent hors ligne avec une instance locale Ollama ou LM Studio.

Quelles plateformes sont prises en charge ?

Linux est la principale plateforme de développement. Windows et macOS sont pris en charge via platform_utils.py. L'application compagnon Android se connecte via WebSocket à l'application de bureau.

Pouvez-vous ajouter une fonctionnalité ou intégrer à notre système ?

Des intégrations personnalisées sont prévues. Contactez-nous avec vos besoins : les demandes courantes incluent des fournisseurs d'IA personnalisés, des flux de travail comptables sur mesure et l'automatisation Tailscale ACL. Conditions sous réserve de finalisation.

Les clés API sont-elles sûres ?

Les clés API sont stockées dans api_keys.env sur votre machine locale et jamais transmis ailleurs que chez le fournisseur que vous choisissez. Le serveur API DB en lecture seule (port 8769) est limité à votre réseau Tailscale.
