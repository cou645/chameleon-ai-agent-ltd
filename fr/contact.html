Foire aux questions

Puis-je essayer Caméléon ?

The product is currently in testing and not yet available for public release. Contact us to register interest or request a demo — we'll be in touch when it's ready.

Est-ce que ça marche sans connexion internet ?

Most features work offline: the editor, accounting, presenter and the driver porter all run locally. AI features require an internet connection for cloud providers (DeepSeek, OpenAI, etc.) but work offline with a local Ollama or LM Studio instance.

Quelles plateformes sont prises en charge ?

Linux is the primary development platform. Windows and macOS are supported via platform_utils.py. The Android companion app connects over WebSocket to the desktop application.

Pouvez-vous ajouter une fonctionnalité ou intégrer à notre système ?

Custom integrations are planned. Contact us with your requirements — common requests include custom AI providers, bespoke accounting workflows and Tailscale ACL automation. Terms subject to finalisation.

Are API keys safe?

API keys are stored in api_keys.env on your local machine and never transmitted anywhere other than the provider you choose. The read-only DB API server (port 8769) is restricted to your Tailscale network.
