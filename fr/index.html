Plateforme d'IA caméléon

Une suite bureautique complète alimentée par l'IA : IDE, présentateur, comptabilité, salle de classe et plus encore. Conçu pour les indépendants et les petites entreprises.

Activement développé 🇫🇷 Affaires au Royaume-Uni 🔒 Le local d'abord / La confidentialité d'abord 🐧 Linux · Windows · macOS

Une seule plateforme. Sept disciplines.

Chameleon AI est une application de bureau PyQt6 créée par Marcos alias stemsee pour Chameleon AI Technicians Ltd. Elle regroupe un éditeur de code d'IA professionnel, un système de présentation inspiré du théâtre, une comptabilité complète en partie double au Royaume-Uni, des salles de classe vidéo d'enseignement des langues, un réseau P2P et bien plus encore, le tout fonctionnant localement sur votre machine sans dépendance au cloud. L'IA IDE intègre Claude Code exécuté dans un terminal (pty) pour l'édition directe de code pilotée par l'IA.

Les fournisseurs d'IA sont échangeables au moment de l'exécution : DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot et plus encore. Les clés ne quittent jamais votre appareil.

Points forts de la plateforme

🔒

Confidentialité locale d'abord

Toutes les données (grands livres, historique des discussions, fichiers de projet) résident dans ~/.chameleon/ sur votre machine. Rien n'est envoyé à un serveur tiers, sauf si vous choisissez explicitement un fournisseur d'IA cloud.

🔄

Fournisseurs d'IA échangeables

Basculez entre DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot et plus en un clic. Exécutez des modèles locaux via Ollama ou LM Studio sans frais.

📱

Compagnon mobile

Le pont Android sur WebSocket relaie le chat AI, les captures d'écran et les téléchargements de fichiers depuis votre téléphone. Contrôlez le présentateur, déclenchez des enregistrements et approuvez les soumissions à distance.

🌐

Maillage de queue

Vidéo et voix peer-to-peer sur votre propre réseau Tailscale. Gérez les ACL, l'autorisation des appareils et les clés directement depuis l'EDI.

🎙️

Voix entrante, voix sortante

Synthèse parole-texte basée sur le murmure pour un codage mains libres. Piper TTS lit à haute voix les réponses de l'IA. Dictez naturellement du code, de la documentation ou des messages.

🐧

Multiplateforme

Fonctionne sous Linux, Windows et macOS. Les chemins spécifiques à la plate-forme, les lecteurs audio et les binaires TTS sont résumés dans platform_utils.py - une base de code partout.
