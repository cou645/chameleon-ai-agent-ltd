🛡 Référence du code de communication sur le terrain

Logistique 4×4 · Édition 1 · 2026 · Chameleon AI Technicians Ltd

Un protocole de code compact à utiliser lorsque la connectivité des données n'est pas disponible. Conçu pour les chauffeurs de camion, les randonneurs et les services d'urgence opérant au-delà des frontières ou sur des terrains éloignés. Trois profils d'utilisateurs partagent un vocabulaire d'alerte et de système commun, permettant une communication inter-agences même dans des conditions gravement dégradées.

Le partage de cette référence est encouragé. L'interopérabilité dépend de la connaissance des codes par toutes les parties.

Enregistrez cette page au format PDF à l'aide de la fonction Imprimer de votre navigateur : Fichier → Imprimer → Enregistrer au format PDF. Conservez une copie sur votre téléphone pour référence hors ligne.

Comment fonctionnent les codes

Chaque code est un lettre (catégorie) suivi de deux chiffres (gravité ou sous-type). Dans chaque catégorie, les nombres inférieurs signifient normal ou mineur, des nombres plus élevés signifient grave ou critique. Le code 99 est toujours le plus sévère.

Un 1 1
│  └─┴── Gravité  11 = nominal → 99 = urgence
└────── Lettre de catégorie

Plusieurs codes peuvent être combinés dans un seul message. Une note en texte libre peut être annexée pour plus de détails.

Conducteur Opérateur camion/transfrontalier
H
Promeneur Membre d'une équipe de plein air / trekking
X
Services d'urgence SAR / ambulancier / ranger / pompier

Codes des services d’urgence – préfixe X
SAR, ambulanciers paramédicaux, équipes de pompiers, rangers – opérations de communication dégradées
Aligné sur les normes internationales d’urgence La catégorie P (triage) utilise l'échelle de priorité internationale P1 à P4 (P1 = Immédiat, P2 = Différé, P3 = Mineur, P4 = Attendant). Les catégories I (incident) et N (décompte) reflètent le cadre de déclaration d'incident majeur MÉTHANE utilisé par les services communs de secours.
