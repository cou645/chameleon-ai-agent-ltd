Платформа Chameleon AI

Повний пакет настільних комп’ютерів на основі штучного інтелекту — IDE, презентатор, бухгалтерія, торгівля, клас тощо. Створено для фрілансерів і малого бізнесу.

Активно розвивається 🇬🇧 Бізнес Великобританії 🔒 Насамперед локальне / Конфіденційність насамперед 🐧 Linux · Windows · macOS

Одна платформа. Вісім дисциплін.

Chameleon AI — це настільна програма PyQt6, розроблена Маркосом, відомим як stemsee, для Chameleon AI Technicians Ltd. Вона включає в себе професійний редактор коду AI, систему презентацій, натхненну театром, повну систему подвійного обліку у Великій Британії, криптовалютну торгівлю Binance, відеокласи для навчання мов, мережу P2P тощо — усе це працює локально на вашому комп’ютері без залежності від хмари. IDE AI інтегрує Claude Code, що працює в терміналі (pty), для прямого редагування коду за допомогою штучного інтелекту.

Постачальники штучного інтелекту можна замінити під час виконання: DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot тощо. Ключі ніколи не залишають ваш пристрій.

Основні моменти платформи

🔒

Конфіденційність на першому місці

Усі дані — бухгалтерські книги, історія чатів, торгові позиції — зберігаються ~/.chameleon/ на вашій машині. Нічого не надсилається на сторонній сервер, якщо ви явно не виберете хмарного постачальника AI.

🔄

Замінні постачальники AI

Перемикайтеся між DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot тощо в один клік. Запускайте локальні моделі через Ollama або LM Studio без будь-яких витрат.

📱

Мобільний компаньйон

Міст Android через WebSocket передає чат AI, знімки екрана та завантаження файлів із вашого телефону. Керуйте доповідачем, запускайте записи та затверджуйте подання віддалено.

🌐

Хвостова сітка

Однорангове відео та голос через власну мережу Tailscale. Керуйте списками контролю доступу, авторизацією пристрою та ключами безпосередньо з IDE.

🎙️

Голос вхід, голос

Перетворення мовлення в текст на основі шепоту для кодування без використання рук. Piper TTS читає вголос відповіді ШІ. Диктуйте код, документацію або повідомлення природним шляхом.

🐧

Кросплатформенність

Працює на Linux, Windows і macOS. Шляхи до певної платформи, аудіопрогравачі та двійкові файли TTS є абстрагованими platform_utils.py — скрізь одна кодова база.
