Часті запитання

Можна спробувати Chameleon?

Наразі продукт тестується і ще не доступний для загального випуску. Зв’яжіться з нами, щоб зареєструвати інтерес або подати запит на демонстрацію — ми зв’яжемося, коли вона буде готова.

Чи працює без підключення до Інтернету?

Більшість функцій працюють в автономному режимі: редактор, бухгалтерія, доповідач і портер водія працюють локально. Функції штучного інтелекту вимагають підключення до Інтернету для хмарних провайдерів (DeepSeek, OpenAI тощо), але працюють в автономному режимі з локальним екземпляром Ollama або LM Studio.

Які платформи підтримуються?

Linux є основною платформою розробки. Windows і macOS підтримуються через platform_utils.py. Супутня програма Android підключається через WebSocket до настільної програми.

Чи можете ви додати функцію або інтегруватися з нашою системою?

Планується індивідуальна інтеграція. Зв’яжіться з нами та повідомте свої вимоги — типові запити включають користувальницькі постачальники штучного інтелекту, індивідуальні облікові робочі процеси та автоматизацію Tailscale ACL. Умови підлягають уточненню.

Чи безпечні ключі API?

Ключі API зберігаються в api_keys.env на вашому локальному комп’ютері та ніколи не передається нікуди, крім обраного вами постачальника. Сервер API БД лише для читання (порт 8769) обмежено вашою мережею Tailscale.

Готовий поговорити?

Надішліть нам електронний лист, і ми зв’яжемося з вами протягом одного робочого дня.
