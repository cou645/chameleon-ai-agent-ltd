よくある質問

カメレオンを試してみてもいいですか？

この製品は現在テスト中であり、まだ一般公開されていません。ご興味を登録するか、デモをリクエストするには、お問い合わせください。準備ができ次第、ご連絡させていただきます。

インターネット接続なしでも動作しますか?

ほとんどの機能はオフラインで動作します。エディター、アカウンティング、プレゼンター、ドライバー ポーターはすべてローカルで実行されます。 AI 機能にはクラウド プロバイダー (DeepSeek、OpenAI など) のインターネット接続が必要ですが、ローカルの Ollama または LM Studio インスタンスを使用してオフラインで動作します。

どのプラットフォームがサポートされていますか?

Linux は主要な開発プラットフォームです。 Windows と macOS は次の方法でサポートされています。 platform_utils.py。 Android コンパニオン アプリは、WebSocket 経由でデスクトップ アプリケーションに接続します。

機能を追加したり、システムと統合したりできますか?

カスタム統合が計画されています。要件については、お問い合わせください。一般的なリクエストには、カスタム AI プロバイダー、オーダーメイドの会計ワークフロー、Tailscale ACL 自動化などが含まれます。条件は最終決定の対象となります。

API キーは安全ですか?

API キーは次の場所に保存されます api_keys.env ローカル マシン上に保存され、選択したプロバイダー以外には決して送信されません。読み取り専用 DB API サーバー (ポート 8769) は、Tailscale ネットワークに制限されています。

話す準備はできていますか?

メールでお問い合わせください。1 営業日以内にご連絡いたします。
