The AI Command Bridge

AI doesn't just chat — it operates the editor through a structured JSON protocol

📖

Διαβάστε μια φορά, επεξεργαστείτε ελεύθερα

The AI reads a file once with get_file_content, κρατά το περιεχόμενο στο πλαίσιο και, στη συνέχεια, εκδίδει πολλαπλές επεξεργασίες που βασίζονται σε άγκυρα χωρίς να διαβάζει ξανά μεταξύ κάθε αλλαγής. Χωρίς σπατάλη μετ' επιστροφής.

🎯

Αντικατάσταση_κειμένου που βασίζεται σε άγκυρα

Επεξεργάζεται το ακριβές κείμενο αναφοράς για εύρεση, όχι αριθμούς γραμμών. Οι αριθμοί γραμμών αλλάζουν καθώς προστίθεται ή αφαιρείται κώδικας — οι αγκυρώσεις όχι. Πολλαπλούς replace_text Οι κλήσεις μπορούν να ομαδοποιηθούν σε μία μόνο απόκριση, όλες εφαρμόζονται ατομικά.

📝

write_file για πλήρη επανεγγραφή

For new modules or complete rewrites the AI sends the entire file content in one command. The system automatically creates a .bak δημιουργία αντιγράφων ασφαλείας πριν από την αντικατάσταση και ανοίγει το αποτέλεσμα σε μια νέα καρτέλα επεξεργασίας.

💻

Ενοποίηση τερματικού

The AI can run shell commands and receive the JSON result before continuing. It waits for each result, handles exit codes, and distinguishes diagnostic, configuration, destructive and session-ending commands automatically.

🛡️

Προστασία έγχυσης

Όλο το περιεχόμενο που επιστρέφεται από αρχεία, τερματικά και ιστοσελίδες είναι τυλιγμένο [UNTRUSTED_CONTENT] markers. The AI is instructed to treat everything inside as data — never instructions — preventing prompt-injection attacks from malicious file content.

Χαρακτηριστικά επεξεργασίας

🗂️

Επεξεργαστής κώδικα πολλαπλών καρτελών

Open multiple files simultaneously. Each tab tracks its own undo history, syntax highlighting and AI command context.

🔀

Ενσωματωμένη προβολή διαφορών

One click to compare the current editor state against the last saved version. Colour-coded unified diff with add/remove highlighting. Exit diff mode to return to full editing.

💬

Streaming AI chat

Οι απαντήσεις μεταδίδονται διακριτικά προς διακριτικό στον πίνακα συνομιλίας. Οι εντολές IDE εξάγονται και εκτελούνται σε ένα νήμα φόντου, έτσι ώστε η διεπαφή χρήστη να παραμένει αποκριτική καθ' όλη τη διάρκεια.

🗃️

Αρθρωτό σύστημα προτροπής

System prompts are plain-text module files in ~/.chameleon/prompts/. Stack any combination: Tech Repair + Academic + Chat. Live token count and cost estimate per provider.

📜

DB ιστορικού εντολών

Every AI command is logged to history.db with session ID, timestamp and result. Full audit trail for every file change the AI made.

🌳

Πρόγραμμα περιήγησης αρχείων

Project tree panel for fast navigation. The AI uses list_project_files και list_open_files για να ανακαλύψει πραγματικές διαδρομές — δεν εφευρίσκει ποτέ τοποθεσίες αρχείων.

Φωνή & Λόγος

🎙️

Whisper speech-to-text

Local Whisper (whisper.cpp) transcribes your voice into the chat input. Hands-free coding — dictate questions, commands or whole prompts without touching the keyboard.

whisper.cppΤοπικό συμπέρασμαΧωρίς σύννεφο STT
🔊

Piper text-to-speech

AI responses are read aloud using Piper TTS — fast, local, natural-sounding voices. Ideal for accessibility or working with your hands occupied. Platform paths resolved automatically on Linux, Windows and macOS.

Σταθμός Διοίκησης Εταιρείας (CCS)

Εκτελέστε το Chameleon IDE ως το κέντρο γραφείου για τον στόλο Logistics 4×4 — δεν χρειάζεται ξεχωριστό λογισμικό αποστολής

🗺

Push διαδρομές και POI στους οδηγούς

Αποστολή α .gpx αρχείο από το IDE απευθείας στο τηλέφωνο οποιουδήποτε προγράμματος οδήγησης μέσω του ομότιμου καναλιού. Οι διαδρομές φορτώνονται αμέσως στον ζωντανό χάρτη του οδηγού. Αυτόνομα σύνολα POI — στάσεις καυσίμου, κίνδυνοι, διευθύνσεις παράδοσης — μπορούν να προωθηθούν ανεξάρτητα από οποιαδήποτε διαδρομή και να εμφανίζονται ως καρφίτσες με χρωματική κωδικοποίηση.

📦

Αποστολή δήλωσης & τιμολογίου

Δημιουργήστε ένα δηλωτικό φορτίου ή ετοιμάστε ένα τιμολόγιο στο IDE και, στη συνέχεια, σπρώξτε το peer-to-peer στο τηλέφωνο του οδηγού. Το πρόγραμμα οδήγησης βλέπει το μανιφέστο προφορτωμένο — χωρίς σάρωση, χωρίς μη αυτόματη καταχώριση. Τα τιμολόγια μπορούν να υπογραφούν στην οθόνη και να επιστραφούν μέσω του ίδιου καναλιού.

🚨

Εκπομπή σε όλο τον στόλο

Στείλτε μια ειδοποίηση προτεραιότητας — Έκτακτη ανάγκη, Σημαντικό, Οδηγίες ή Πληροφορίες — και θα εμφανίζεται ταυτόχρονα ως γραμμή ελέγχου πλήρους πλάτους στην οθόνη κάθε οδηγού. Το γραφείο διατηρεί πλήρως την προσοχή του στόλου χωρίς τηλεφώνημα σε κάθε όχημα.

📡

Ενσωματωμένος διακομιστής σήματος — χωρίς ρελέ σύννεφο

Ο διακομιστής σημάτων Chameleon (chameleon_signal.py) runs directly on the office PC alongside the IDE, or inside Termux on any Android phone in the fleet. Fleet phones connect to it over Tailscale or the local hotspot — no third-party relay, no monthly server bill.
