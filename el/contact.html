Συχνές Ερωτήσεις

Μπορώ να δοκιμάσω το Chameleon;

Το προϊόν βρίσκεται επί του παρόντος σε δοκιμή και δεν είναι ακόμη διαθέσιμο για δημόσια κυκλοφορία. Επικοινωνήστε μαζί μας για να δηλώσετε ενδιαφέρον ή να ζητήσετε μια επίδειξη — θα επικοινωνήσουμε όταν είναι έτοιμο.

Λειτουργεί χωρίς σύνδεση στο διαδίκτυο;

Οι περισσότερες λειτουργίες λειτουργούν εκτός σύνδεσης: ο επεξεργαστής, η λογιστική, ο παρουσιαστής και ο οδηγός αχθοφόρος εκτελούνται όλα τοπικά. Οι λειτουργίες AI απαιτούν σύνδεση στο Διαδίκτυο για παρόχους cloud (DeepSeek, OpenAI, κ.λπ.), αλλά λειτουργούν εκτός σύνδεσης με μια τοπική παρουσία Olama ή LM Studio.

Ποιες πλατφόρμες υποστηρίζονται;

Το Linux είναι η κύρια πλατφόρμα ανάπτυξης. Τα Windows και το macOS υποστηρίζονται μέσω platform_utils.py. Η συνοδευτική εφαρμογή Android συνδέεται μέσω WebSocket στην εφαρμογή για υπολογιστές.

Μπορείτε να προσθέσετε μια δυνατότητα ή να ενσωματώσετε το σύστημά μας;

Προγραμματίζονται προσαρμοσμένες ενσωματώσεις. Επικοινωνήστε μαζί μας για τις απαιτήσεις σας — τα κοινά αιτήματα περιλαμβάνουν προσαρμοσμένους παρόχους τεχνητής νοημοσύνης, προσαρμοσμένες ροές εργασιών λογιστικής και αυτοματισμό Tailscale ACL. Όροι που υπόκεινται σε οριστικοποίηση.

Είναι ασφαλή τα κλειδιά API;

Τα κλειδιά API αποθηκεύονται σε api_keys.env στον τοπικό σας υπολογιστή και δεν μεταδόθηκε ποτέ πουθενά αλλού εκτός από τον πάροχο που επιλέγετε. Ο διακομιστής DB API μόνο για ανάγνωση (θύρα 8769) περιορίζεται στο δίκτυό σας Tailscale.

Έτοιμοι να μιλήσουμε;

Στείλτε μας ένα email και θα επικοινωνήσουμε μαζί σας εντός μιας εργάσιμης ημέρας.
