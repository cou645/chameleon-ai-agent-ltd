Domande frequenti

Posso provare Camaleonte?

Il prodotto è attualmente in fase di test e non è ancora disponibile per il rilascio pubblico. Contattaci per manifestare il tuo interesse o richiedere una demo: ti contatteremo quando sarà pronta.

Funziona senza connessione internet?

La maggior parte delle funzionalità funziona offline: l'editor, la contabilità, il presentatore e l'autista porter funzionano tutti localmente. Le funzionalità AI richiedono una connessione Internet per i provider cloud (DeepSeek, OpenAI, ecc.) ma funzionano offline con un'istanza locale Ollama o LM Studio.

Quali piattaforme sono supportate?

Linux è la piattaforma di sviluppo principale. Windows e macOS sono supportati tramite platform_utils.py. L'app complementare Android si connette tramite WebSocket all'applicazione desktop.

Puoi aggiungere una funzionalità o integrarla con il nostro sistema?

Sono previste integrazioni personalizzate. Contattaci con le tue esigenze: le richieste comuni includono fornitori di intelligenza artificiale personalizzati, flussi di lavoro contabili su misura e automazione ACL Tailscale. Termini soggetti a finalizzazione.

Le chiavi API sono sicure?

Le chiavi API sono archiviate in api_keys.env sul tuo computer locale e mai trasmesso da nessun'altra parte se non presso il provider da te scelto. Il server API DB di sola lettura (porta 8769) è limitato alla tua rete Tailscale.
