Często zadawane pytania

Czy mogę spróbować Chameleona?

Produkt jest obecnie w fazie testów i nie jest jeszcze dostępny do publicznego wydania. Skontaktuj się z nami, aby zgłosić zainteresowanie lub zamówić wersję demonstracyjną — skontaktujemy się z Tobą, gdy będzie gotowa.

Czy to działa bez połączenia z Internetem?

Większość funkcji działa w trybie offline: edytor, księgowość, prezenter i porter sterowników działają lokalnie. Funkcje AI wymagają połączenia z Internetem w przypadku dostawców usług w chmurze (DeepSeek, OpenAI itp.), ale działają w trybie offline z lokalną instancją Ollama lub LM Studio.

Jakie platformy są obsługiwane?

Linux jest podstawową platformą programistyczną. Obsługiwane są systemy Windows i macOS za pośrednictwem platform_utils.py. Aplikacja towarzysząca dla systemu Android łączy się za pośrednictwem protokołu WebSocket z aplikacją komputerową.

Czy możesz dodać funkcję lub zintegrować się z naszym systemem?

Planowane są niestandardowe integracje. Skontaktuj się z nami, aby przedstawić swoje wymagania — częste prośby obejmują niestandardowych dostawców sztucznej inteligencji, niestandardowe przepływy pracy księgowej i automatyzację listy ACL Tailscale. Warunki podlegają finalizacji.

Czy klucze API są bezpieczne?

Klucze API są przechowywane w api_keys.env na Twoim komputerze lokalnym i nigdy nie są przesyłane nigdzie indziej niż u wybranego przez Ciebie dostawcy. Serwer DB API tylko do odczytu (port 8769) jest ograniczony do Twojej sieci Tailscale.
