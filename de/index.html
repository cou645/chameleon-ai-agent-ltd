Chameleon AI-Plattform

Eine komplette KI-gestützte Desktop-Suite – IDE, Präsentator, Buchhaltung, Klassenzimmer und mehr. Entwickelt für Freiberufler und kleine Unternehmen.

Aktiv entwickelt 🇬🇧 Britisches Geschäft 🔒 Lokal zuerst / Datenschutz zuerst 🐧 Linux · Windows · macOS

Eine Plattform. Sieben Disziplinen.

Chameleon AI ist eine PyQt6-Desktopanwendung, die von Marcos alias stemsee für Chameleon AI Technicians Ltd. entwickelt wurde. Sie bündelt einen professionellen AI-Code-Editor, ein vom Theater inspiriertes Präsentationssystem, eine vollständige doppelte Buchführung in Großbritannien, Sprachunterrichts-Videoklassenzimmer, P2P-Netzwerke und mehr – alles läuft lokal auf Ihrem Computer ohne Cloud-Abhängigkeit. Die AI-IDE integriert Claude Code, der in einem Terminal (pty) ausgeführt wird, für die direkte KI-gesteuerte Codebearbeitung.

KI-Anbieter sind zur Laufzeit austauschbar: DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot und mehr. Schlüssel verlassen niemals Ihr Gerät.

Plattform-Highlights

🔒

Privatsphäre an erster Stelle vor Ort

Alle Daten – Hauptbücher, Chat-Verlauf, Projektdateien – leben darin ~/.chameleon/ auf Ihrer Maschine. Es wird nichts an einen Server eines Drittanbieters gesendet, es sei denn, Sie wählen ausdrücklich einen Cloud-KI-Anbieter aus.

🔄

Austauschbare KI-Anbieter

Wechseln Sie mit einem Klick zwischen DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot und mehr. Führen Sie lokale Modelle kostenlos über Ollama oder LM Studio aus.

📱

Mobiler Begleiter

Die Android-Brücke über WebSocket leitet KI-Chat, Screenshots und Datei-Uploads von Ihrem Telefon weiter. Steuern Sie den Moderator, lösen Sie Aufzeichnungen aus und genehmigen Sie Einsendungen aus der Ferne.

🌐

Tailscale-Netz

Peer-to-Peer-Video und -Sprache über Ihr eigenes Tailscale-Netzwerk. Verwalten Sie ACLs, Geräteautorisierung und Schlüssel direkt über die IDE.

🎙️

Stimme rein, Stimme raus

Flüsterbasierte Sprachausgabe für freihändiges Codieren. Piper TTS liest KI-Antworten laut vor. Diktieren Sie Code, Dokumentation oder Nachrichten auf natürliche Weise.

🐧

Plattformübergreifend

Läuft unter Linux, Windows und macOS. Plattformspezifische Pfade, Audioplayer und TTS-Binärdateien werden abstrahiert platform_utils.py – eine Codebasis überall.
