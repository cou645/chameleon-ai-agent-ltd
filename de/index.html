A complete AI-powered desktop suite — IDE, presenter, accounting, trading, classroom and more. Built for freelancers and small businesses.

Aktiv entwickelt 🇬🇧 Britisches Geschäft 🔒 Lokal zuerst / Datenschutz zuerst 🐧 Linux · Windows · macOS

Eine Plattform. Acht Disziplinen.

Chameleon AI is a PyQt6 desktop application built by Marcos aka stemsee for Chameleon AI Technicians Ltd. It bundles a professional AI code editor, a theatre-inspired presentation system, full UK double-entry accounting, Binance crypto trading, language-teaching video classrooms, P2P networking and more — all running locally on your machine with no cloud dependency. The AI IDE integrates Claude Code running in a terminal (pty) for direct AI-driven code editing.

AI providers are swappable at runtime: DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot and more. Keys never leave your device.

Plattform-Highlights

🔒

Privatsphäre an erster Stelle vor Ort

Alle Daten – Hauptbücher, Chat-Verlauf, Handelspositionen – leben darin ~/.chameleon/ on your machine. Nothing is sent to a third-party server unless you explicitly choose a cloud AI provider.

🔄

Swappable AI providers

Switch between DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot and more in one click. Run local models via Ollama or LM Studio with zero cost.

📱

Mobiler Begleiter

Android bridge over WebSocket relays AI chat, screenshots and file uploads from your phone. Control the presenter, trigger recordings and approve submissions remotely.

🌐

Tailscale Mesh

Peer-to-peer video and voice over your own Tailscale network. Manage ACLs, device authorisation and keys directly from the IDE.

🎙️

Stimme rein, Stimme raus

Whisper-based speech-to-text for hands-free coding. Piper TTS reads AI responses aloud. Dictate code, documentation or messages naturally.

🐧

Plattformübergreifend

Runs on Linux, Windows and macOS. Platform-specific paths, audio players and TTS binaries are abstracted in platform_utils.py – eine Codebasis überall.
