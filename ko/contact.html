자주 묻는 질문

카멜레온을 시험해 볼 수 있나요?

해당 제품은 현재 테스트 중이며 아직 공개 출시되지 않았습니다. 관심을 등록하거나 데모를 요청하려면 저희에게 연락하세요. 준비가 되면 연락드리겠습니다.

인터넷에 연결하지 않고도 작동합니까?

대부분의 기능은 오프라인으로 작동합니다. 편집자, 회계, 발표자 및 드라이버 포터는 모두 로컬에서 실행됩니다. AI 기능을 사용하려면 클라우드 공급자(DeepSeek, OpenAI 등)에 대한 인터넷 연결이 필요하지만 로컬 Ollama 또는 LM Studio 인스턴스를 사용하여 오프라인으로 작동합니다.

어떤 플랫폼이 지원되나요?

Linux는 기본 개발 플랫폼입니다. Windows 및 macOS는 다음을 통해 지원됩니다. platform_utils.py. Android 컴패니언 앱은 WebSocket을 통해 데스크톱 애플리케이션에 연결됩니다.

기능을 추가하거나 우리 시스템과 통합할 수 있나요?

맞춤형 통합이 계획되어 있습니다. 귀하의 요구 사항이 있으면 당사에 문의하십시오. 일반적인 요청에는 맞춤형 AI 제공업체, 맞춤형 회계 워크플로 및 Tailscale ACL 자동화가 포함됩니다. 확정될 조건.

API 키는 안전한가요?

API 키는 다음 위치에 저장됩니다. api_keys.env 귀하의 로컬 컴퓨터에 저장되며 귀하가 선택한 공급자 이외의 다른 곳으로는 전송되지 않습니다. 읽기 전용 DB API 서버(포트 8769)는 Tailscale 네트워크로 제한됩니다.

이야기할 준비가 되셨나요?
