KameleonStack

The universal content container. Each stack holds multiple slots (images, text, PDFs). Two modes:

  • 콘텐츠 — 교사가 제어하는 ​​슬라이드
  • 단계 — 동료 푸시 제출을 허용합니다.
🎬

애니메이션 시스템

자유형 모션 경로가 있는 개체별 들어오기/나가기 애니메이션입니다. 경로는 저장 전에 Douglas-Peucker 압축과 Catmull-Rom 스플라인 보간을 통해 부드럽게 화면에 그려집니다.

🗃️

드레서 패널

무대 밖 대기 공간 - 옷장에서 개체를 만들고 더블 클릭으로 무대에서 미리 볼 수 있습니다. 마우스 오른쪽 버튼 클릭 메뉴에서 다시 Dresser로 드래그하세요. 속성 패널에는 애니메이션 시작/종료 상태가 표시됩니다.

🔢

번호가 매겨진 신호

트리거 포인트는 연극 조명 신호처럼 프리젠테이션을 단계별로 진행시킵니다. 각 큐는 개체를 안팎으로 애니메이션화하거나 스택 슬롯을 전환하거나 시간 제한 시퀀스를 실행할 수 있습니다.

💾

.kpt 프로젝트 번들

프로젝트는 ZIP 아카이브(.kpt) containing all media assets, layout data and animation configs. Open on any machine running Chameleon.

📐

논리적 좌표

스테이지는 디스플레이 해상도와 관계없이 0~10,000의 논리적 좌표 공간을 사용합니다. 프레젠테이션은 노트북 화면이나 4K 프로젝터에서 동일하게 보입니다.

모바일 및 피어 제어

휴대폰에서 발표자를 실행하세요. 학생들이 자신의 콘텐츠를 푸시하도록 허용

📱

Android mobile bridge

The Chameleon mobile bridge (WebSocket) relays theatre commands from any Android device. Control stack_next / 이전 / 재생 / 일시 중지 / 중지, 이미지 또는 텍스트 슬롯을 스택에 푸시하고 실시간으로 브로드캐스트되는 스택 상태를 확인하세요.

🙋

동료 제출

Students push images or text to a Stage-mode stack from their phones. Submissions appear in the thumbnail strip with an orange pulsing border. The presenter can 승인, 보류, 거부 또는 승격 화면에 표시되기 전에 각각.

✏️

공유 드로잉 보드

모바일 브릿지에서 제어 가능한 협업 프리핸드 캔버스 — draw_Stroke / 지우기 / 실행 취소. Ideal for annotating slides, brainstorming or live quizzes.

🤖

ADB 전화 미러 및 PPT 캡처

ADB폰박스 streams a live Android screen directly onto the stage via ADB. PPT거울상자 captures a PowerPoint window — present your existing decks inside Kameleon without conversion.
