🛡 Referencia del código de comunicaciones de campo

Logistics 4×4 · Edition 1 · 2026 · Chameleon AI Technicians Ltd

A compact code protocol for use when data connectivity is unavailable. Designed for truck drivers, hikers, and emergency services operating across borders or in remote terrain. Three user profiles share a common alert and system vocabulary — enabling cross-agency communication even under severely degraded conditions.

Se recomienda compartir esta referencia. La interoperabilidad depende de que todas las partes conozcan los códigos.

Cómo funcionan los códigos

Cada código es un carta (categoría) seguida de dos dígitos (gravedad o subtipo). Dentro de cada categoría, los números más bajos significan normal o menor, los números más altos significan grave o critico. El código 99 es siempre el más severo.

Un 1 1
│  └─┴── Gravedad  11 = nominal → 99 = emergencia
└────── Letra de categoría

Multiple codes can be combined in a single message. A free-text note can be appended for additional detail.

Códigos de servicios de emergencia: prefijo X
SAR, paramédicos, equipos de bomberos, guardabosques: operaciones de comunicaciones degradadas
Alineado con los estándares internacionales de emergencia La categoría P (triaje) utiliza la escala de prioridad internacional P1-P4 (P1=Inmediata, P2=Tardía, P3=Menor, P4=Expectante). Las categorías I (incidente) y N (recuento) reflejan el marco de declaración de incidentes mayores de METANO utilizado por los servicios de emergencia conjuntos.
