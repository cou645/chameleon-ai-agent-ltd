Preguntas frecuentes

¿Puedo probar Camaleón?

El producto se encuentra actualmente en prueba y aún no está disponible para su lanzamiento público. Contáctenos para registrar interés o solicitar una demostración; nos comunicaremos con usted cuando esté listo.

¿Funciona sin conexión a Internet?

La mayoría de las funciones funcionan sin conexión: el editor, la contabilidad, el presentador y el conductor portero se ejecutan localmente. Las funciones de IA requieren una conexión a Internet para los proveedores de la nube (DeepSeek, OpenAI, etc.), pero funcionan sin conexión con una instancia local de Ollama o LM Studio.

¿Qué plataformas son compatibles?

Linux es la principal plataforma de desarrollo. Windows y macOS son compatibles a través de platform_utils.py. La aplicación complementaria de Android se conecta a través de WebSocket a la aplicación de escritorio.

¿Puedes agregar una función o integrarla con nuestro sistema?

Se planean integraciones personalizadas. Contáctenos con sus requisitos: las solicitudes comunes incluyen proveedores de IA personalizados, flujos de trabajo de contabilidad personalizados y automatización Tailscale ACL. Condiciones sujetas a finalización.

¿Son seguras las claves API?

Las claves API se almacenan en api_keys.env en su máquina local y nunca se transmite a ningún otro lugar que no sea el proveedor que elija. El servidor DB API de solo lectura (puerto 8769) está restringido a su red Tailscale.

¿Listo para hablar?

Envíenos un correo electrónico y nos comunicaremos con usted dentro de un día hábil.
