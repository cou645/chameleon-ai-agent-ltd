AI-Powered IDE

A full-featured PyQt6 code editor where AI doesn't just suggest — it reads, writes, runs and verifies code on your behalf through a structured command bridge.

Multi-provider AI 🎙️ वॉयस इनपुट और टीटीएस 🔀 अंतर दृश्य ⚡ स्ट्रीमिंग प्रतिक्रियाएँ

The AI Command Bridge

AI doesn't just chat — it operates the editor through a structured JSON protocol

📖

एक बार पढ़ें, स्वतंत्र रूप से संपादित करें

The AI reads a file once with get_file_content, सामग्री को संदर्भ में रखता है, फिर प्रत्येक परिवर्तन के बीच दोबारा पढ़े बिना कई एंकर-आधारित संपादन जारी करता है। कोई व्यर्थ चक्कर नहीं।

🎯

एंकर-आधारित रिप्लेस_टेक्स्ट

ढूँढ़ने के लिए संदर्भ सटीक पाठ संपादित करता है, पंक्ति संख्याएँ नहीं। जैसे ही कोड जोड़ा या हटाया जाता है, लाइन नंबर बदल जाते हैं - एंकर नहीं। विभिन्न replace_text कॉल को एक ही प्रतिक्रिया में बैच किया जा सकता है, सभी को परमाणु रूप से लागू किया जा सकता है।

📝

पूर्ण पुनर्लेखन के लिए write_file

For new modules or complete rewrites the AI sends the entire file content in one command. The system automatically creates a .bak ओवरराइटिंग से पहले बैकअप लें और परिणाम को एक नए संपादक टैब में खोलें।

💻

टर्मिनल एकीकरण

The AI can run shell commands and receive the JSON result before continuing. It waits for each result, handles exit codes, and distinguishes diagnostic, configuration, destructive and session-ending commands automatically.

🛡️

इंजेक्शन सुरक्षा

फ़ाइलों, टर्मिनलों और वेब पेजों से लौटाई गई सभी सामग्री लपेटी गई है [UNTRUSTED_CONTENT] markers. The AI is instructed to treat everything inside as data — never instructions — preventing prompt-injection attacks from malicious file content.

संपादक सुविधाएँ

🗂️

मल्टी-टैब कोड संपादक

Open multiple files simultaneously. Each tab tracks its own undo history, syntax highlighting and AI command context.

🔀

अंतर्निहित अंतर दृश्य

One click to compare the current editor state against the last saved version. Colour-coded unified diff with add/remove highlighting. Exit diff mode to return to full editing.

Streaming AI chat

🗃️

मॉड्यूलर प्रॉम्प्ट सिस्टम

System prompts are plain-text module files in ~/.chameleon/prompts/. Stack any combination: Tech Repair + Academic + Chat. Live token count and cost estimate per provider.

📜

कमांड इतिहास डीबी

Every AI command is logged to history.db with session ID, timestamp and result. Full audit trail for every file change the AI made.

🌳

फ़ाइल ब्राउज़र

Project tree panel for fast navigation. The AI uses list_project_files और list_open_files वास्तविक पथ खोजने के लिए - यह कभी भी फ़ाइल स्थानों का आविष्कार नहीं करता है।

आवाज़ और वाणी

🎙️

Whisper speech-to-text

Local Whisper (whisper.cpp) transcribes your voice into the chat input. Hands-free coding — dictate questions, commands or whole prompts without touching the keyboard.

AI responses are read aloud using Piper TTS — fast, local, natural-sounding voices. Ideal for accessibility or working with your hands occupied. Platform paths resolved automatically on Linux, Windows and macOS.

कंपनी कमांड स्टेशन (सीसीएस)

अपने लॉजिस्टिक्स 4×4 बेड़े के लिए गिरगिट आईडीई को कार्यालय केंद्र के रूप में चलाएं - किसी अलग डिस्पैच सॉफ़्टवेयर की आवश्यकता नहीं है

🗺

ड्राइवरों को मार्ग और पीओआई भेजें

भेजें एक .gpx पीयर चैनल पर आईडीई से सीधे किसी भी ड्राइवर के फोन पर फाइल करें। ड्राइवर के लाइव मैप पर रूट तुरंत लोड हो जाते हैं। स्टैंडअलोन POI सेट - ईंधन रोकना, खतरे, वितरण पते - किसी भी मार्ग से स्वतंत्र रूप से धकेले जा सकते हैं और रंग-कोडित पिन के रूप में दिखाई दे सकते हैं।

📦

घोषणापत्र एवं चालान प्रेषण

एक कार्गो मेनिफेस्ट बनाएं या आईडीई में एक चालान तैयार करें, फिर इसे ड्राइवर के फोन पर पीयर-टू-पीयर पुश करें। ड्राइवर को मैनिफ़ेस्ट पहले से लोडेड दिखाई देता है - कोई स्कैनिंग नहीं, कोई मैन्युअल प्रविष्टि नहीं। चालानों पर ऑन-स्क्रीन हस्ताक्षर किए जा सकते हैं और उसी चैनल पर लौटाए जा सकते हैं।

🚨

फ्लीट-वाइड टिकर प्रसारण

प्राथमिकता अलर्ट भेजें - आपातकालीन, महत्वपूर्ण, निर्देश या जानकारी - और यह प्रत्येक ड्राइवर की स्क्रीन पर एक साथ पूर्ण-चौड़ाई वाले टिकर बार के रूप में दिखाई देता है। कार्यालय प्रत्येक वाहन पर फ़ोन किए बिना बेड़े के ध्यान की पूरी कमान में रहता है।

📡

अंतर्निर्मित सिग्नल सर्वर - कोई क्लाउड रिले नहीं

गिरगिट सिग्नल सर्वर (chameleon_signal.py) runs directly on the office PC alongside the IDE, or inside Termux on any Android phone in the fleet. Fleet phones connect to it over Tailscale or the local hotspot — no third-party relay, no monthly server bill.
