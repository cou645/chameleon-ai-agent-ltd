Perguntas frequentes

Posso experimentar o Camaleão?

O produto está atualmente em testes e ainda não está disponível para lançamento público. Entre em contato conosco para registrar interesse ou solicitar uma demonstração - entraremos em contato quando estiver pronto.

Funciona sem conexão com a internet?

A maioria dos recursos funciona offline: o editor, a contabilidade, o apresentador e o driver porteiro são executados localmente. Os recursos de IA exigem uma conexão com a Internet para provedores de nuvem (DeepSeek, OpenAI, etc.), mas funcionam offline com uma instância local do Ollama ou LM Studio.

Quais plataformas são suportadas?

Linux é a principal plataforma de desenvolvimento. Windows e macOS são suportados via platform_utils.py. O aplicativo complementar do Android se conecta via WebSocket ao aplicativo de desktop.

Você pode adicionar um recurso ou integrar com nosso sistema?

Integrações personalizadas estão planejadas. Entre em contato conosco e informe suas necessidades – solicitações comuns incluem provedores de IA personalizados, fluxos de trabalho de contabilidade personalizados e automação Tailscale ACL. Termos sujeitos a finalização.

As chaves de API são seguras?

As chaves de API são armazenadas em api_keys.env em sua máquina local e nunca transmitido para qualquer outro lugar que não seja o provedor que você escolher. O servidor DB API somente leitura (porta 8769) está restrito à sua rede Tailscale.
