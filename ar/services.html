الخدمة 2

🌍 Domain Name Configuration

DNS configuration is the part most people find confusing — registrar logins, record types, propagation waits. The AI handles it directly where possible and provides exact step-by-step guidance everywhere else.

  • Identify your registrar automatically or via a short questionnaire
  • قم بإنشاء سجلات DNS الدقيقة المطلوبة (A، AAAA، CNAME) في جدول النسخ واللصق
  • تقديم تعليمات خاصة بالمسجل - Squarespace وGoDaddy وNamecheap وCloudflare و123-reg والمزيد
  • Plain-English explanation of every record type before you touch anything
  • Set custom domain via GitHub Pages API — no manual form entry
  • مراقبة انتشار DNS وتأكيد إصدار شهادة HTTPS
  • Full automation for registrars that provide an API (Cloudflare, Namecheap)
⚠️
لماذا يهم هذا: In our own testing, domain configuration caused the most confusion — a domain had silently migrated from Google Domains to Squarespace with no obvious redirect, and the DNS record types (A, AAAA, CNAME) were completely unfamiliar. The AI now asks your registrar upfront and gives instructions that require no prior DNS knowledge.
