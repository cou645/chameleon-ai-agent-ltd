الأسئلة المتداولة

هل يمكنني تجربة الحرباء؟

المنتج قيد الاختبار حاليًا وليس متاحًا للإصدار العام بعد. اتصل بنا لتسجيل الاهتمام أو طلب عرض توضيحي — سنتواصل معك عندما يصبح جاهزًا.

هل يعمل بدون اتصال بالإنترنت؟

تعمل معظم الميزات دون الاتصال بالإنترنت: يتم تشغيل المحرر والمحاسبة ومقدم العرض وبرنامج التشغيل محليًا. تتطلب ميزات الذكاء الاصطناعي اتصالاً بالإنترنت لموفري الخدمات السحابية (DeepSeek، وOpenAI، وما إلى ذلك) ولكنها تعمل دون اتصال بالإنترنت مع مثيل Ollama أو LM Studio المحلي.

ما هي المنصات المدعومة؟

Linux هو منصة التطوير الأساسية. يتم دعم نظامي التشغيل Windows وmacOS عبر platform_utils.py. يتصل تطبيق Android المصاحب عبر WebSocket بتطبيق سطح المكتب.

هل يمكنك إضافة ميزة أو التكامل مع نظامنا؟

يتم التخطيط لعمليات التكامل المخصصة. اتصل بنا لإبلاغنا بمتطلباتك - تتضمن الطلبات الشائعة موفري الذكاء الاصطناعي المخصصين وسير عمل المحاسبة المخصصة وأتمتة Tailscale ACL. الشروط تخضع للإنهاء.

هل مفاتيح API آمنة؟

يتم تخزين مفاتيح API في api_keys.env على جهازك المحلي ولا يتم نقلها أبدًا إلى أي مكان آخر غير المزود الذي تختاره. يقتصر خادم DB API للقراءة فقط (المنفذ 8769) على شبكة Tailscale الخاصة بك.

على استعداد للحديث؟
