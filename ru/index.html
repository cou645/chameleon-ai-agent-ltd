Платформа искусственного интеллекта Хамелеон

Полный набор настольных компьютеров на базе искусственного интеллекта — IDE, презентатор, бухгалтерский учет, торговля, классная комната и многое другое. Создан для фрилансеров и малого бизнеса.

Активно развивается 🇷🇧 Бизнес в Великобритании 🔒 В первую очередь локальность/в первую очередь конфиденциальность 🐧 Linux · Windows · macOS

Одна платформа. Восемь дисциплин.

Chameleon AI — это настольное приложение PyQt6, созданное Маркосом, также известным как Stemsee, для Chameleon AI Technicians Ltd. Оно включает в себя профессиональный редактор кода искусственного интеллекта, театральную систему презентаций, полную двойную бухгалтерию в Великобритании, криптоторговлю на Binance, видеоклассы по обучению языкам, P2P-сети и многое другое — все это работает локально на вашем компьютере без зависимости от облака. AI IDE интегрирует Claude Code, работающий в терминале (pty), для прямого редактирования кода с помощью AI.

Поставщики ИИ можно заменять во время выполнения: DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot и другие. Ключи никогда не покидают ваше устройство.

Основные характеристики платформы

🔒

Локальная конфиденциальность

Все данные — реестры, история чатов, торговые позиции — хранятся в ~/.chameleon/ на вашей машине. Ничего не отправляется на сторонний сервер, если вы явно не выберете поставщика облачного ИИ.

🔄

Сменные поставщики ИИ

Переключайтесь между DeepSeek, OpenAI, Anthropic, Groq, Ollama, Mistral, Moonshot и другими в один клик. Запускайте локальные модели через Ollama или LM Studio бесплатно.

📱

Мобильный компаньон

Мост Android через WebSocket передает чат AI, снимки экрана и загрузку файлов с вашего телефона. Управляйте докладчиком, запускайте записи и утверждайте материалы удаленно.

🌐

Сетка хвостовой чешуи

Одноранговое видео и голосовая связь через вашу собственную сеть Tailscale. Управляйте списками управления доступом, авторизацией устройств и ключами непосредственно из IDE.

🎙️

Голосовой вход, голосовой выход

Преобразование речи в текст на основе шепота для кодирования без помощи рук. Пайпер TTS зачитывает ответы ИИ вслух. Диктуйте код, документацию или сообщения естественным образом.

🐧

Кросс-платформенный

Работает на Linux, Windows и macOS. Пути, специфичные для платформы, аудиоплееры и двоичные файлы TTS абстрагированы в платформа_utils.py — везде одна кодовая база.
