SERVICE 2

🌍 Domain Name Configuration

DNS configuration is the part most people find confusing — registrar logins, record types, propagation waits. The AI handles it directly where possible and provides exact step-by-step guidance everywhere else.

  • Identify your registrar automatically or via a short questionnaire
  • Generate the exact DNS records required (A, AAAA, CNAME) in a copy-paste table
  • Provide registrar-specific instructions — Squarespace, GoDaddy, Namecheap, Cloudflare, 123-reg and more
  • Plain-English explanation of every record type before you touch anything
  • Set custom domain via GitHub Pages API — no manual form entry
  • Monitor DNS propagation and confirm HTTPS certificate issuance
  • Full automation for registrars that provide an API (Cloudflare, Namecheap)
⚠️
Why this matters: In our own testing, domain configuration caused the most confusion — a domain had silently migrated from Google Domains to Squarespace with no obvious redirect, and the DNS record types (A, AAAA, CNAME) were completely unfamiliar. The AI now asks your registrar upfront and gives instructions that require no prior DNS knowledge.
